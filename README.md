
# MangoRead

MangoRead is a manga catalog project.

 With its categories and genres. You can also comment on the manga, delete and update your own comments.
## Installation



```bash
git clone https://github.com/Marrlenns/MangoRead.git
cd MangoRead
pip install -r requirements.txt

```
    
## Usage

```javascript
./manage.py makemigrations
./manage.py migrate
./manage.py runserver

```
